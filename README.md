## 遗传算法图像分割
# ImageSegmentationBasedOnGeneticAlgorithms
这是上学期期末的课程设计，我的选题是采用matlab代码以及遗传算法来实现图像的分割。  
详细设计了GUI图形界面，以及对比了经典遗传算法以及改进版遗传算法之间的效率区别等等。
